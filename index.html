Wrzucanie Octopressa Na Githuba

Jan 17th, 2012

Aby blog nie siedział tylko na komputerze w kilku krokach możemy wrzycic go na Github Pages.

Aby to zrobić należy posępować według kroków:

  • zakładamy nowe repozytorium na githubie
  • następnie w terminalu przechodzimy do głównego katalogu octopressa
  • tworzymy nową gałąź np. git checkout -b source
  • następnie musisz dodac remote’a na te repo, a dokładnie użyć komendy git remote add origin git@github.com:nazwa_uzytkownika/nazwa_repo
  • a następnie pchnąc na source’a git push origin source
  • po zakończeniu wpisujemy rvm rvmrc trust
  • później przeładowywujemy rvm komendą rvm reload
  • następnie wpisujemy rake setup_github_pages i wklejamy nasz adres repozytorium git@github.com:nazwa_użytkownika/nazwa_repo
  • wpisujemy rake install aby zainstalować na stronie podstawowy motyw
  • wpisujemy rake generate aby wygenerowac posty i strony do publicznego katalogu
  • na koniec wpisujemy rake deploy aby wdrążyć zainstniałe zmiany

Po wykonaniu wszystkich kroków i zmianie wartości root na /nazwa_repo w config.yml octopress powinien pojawić się na stronie nazwa_uzytkownika.github.com/nazwa_repo. Trzeba pamiętać że po zmianie pliku config.yml trzeba znowu użyć komend rake generate i rake deploy.

Instalowanie Octopressa

Jan 17th, 2012

Aby zainstalowac Octopressa:

  1. Zainstaluj conajmniej Ruby 1.9.2 gdyż starszych wersji octopress nie obsługuje. Możesz to zrobić dzieki RVM(Ruby Version Manager) a dokładnie tak:
    • instalujemy rvm z terminala na pomocą bash < <(curl -s https://raw.github.com/wayneeseguin/rvm/master/binscripts/rvm-installer )
    • następnie dodajemy do .basmohprofile linijkę echo '[[ -s "$HOME/.rvm/scripts/rvm" ]] && . "$HOME/.rvm/scripts/rvm" # Load RVM function' >> ~/.bash_profile aby terminal ładował rvm
    • przeładowywujemy bashprofile source .bash_profile, aby zmiany w bashprofile były zauważone
    • type rvm | head -1 sprawdzamy czy rvm się dobrze zainstalował, jeżeli pokaże się “rvm jest funkcją” to znaczy, że rvm dobrze sie zainstalował
    • wpisujemy rvm requirements aby sprawdzić jakie aplikacje są potrzebne do dobrego działania ruby’ego
    • następnie instalujemy wskazane aplikacje do odpowiedniego działania ruby’ego
    • wpisujemy rvm list known aby sprawdzić jakie wersje programów rvm obecnie obsługuje i jakie za pomocą rvm możemy zainstalowac
    • wpisujemy rvm install 1.9.2 aby zainstalować ruby 1.9.2
    • wpisujemy ruby -v, aby sprawdzić czy system operacyjny używa wersji którą mu zainsatalowaliśmy, jeżeli nie odczytuje wersji 1.9.2 którą zainstalowaliśmy należy wpisać rvm use 1.9.2

I takim sposobem zainstalowaliśmy ruby dzieki czemu możemy zacząć instalowac samego “Octopressa”.

  1. Aby zacząć instalację najpierw trzeba go sklonować repozytorium Octopressa ze strony imathisa na githubie.

    • po pobraniu przejdź do folderu octopressa
    • następnie zainstaluj gem bundler abyś mógł zainstalować gemy potrzebne do odpowiedniego działania octopressa
    • następnie użyj komendy rake install aby zainstalowac podstawowy motyw

  2. I teraz mamy zainstalowanego i działającego octopressa. Aby zmienić podstawowe ustawienia octopressa abyś mógł nim zarządzać według własnej woli. Przechodzimy do katalogu głównego octopressa i otwieramy plik _config.yml. W tym pliku możemy zmienić:

    • tytuł
    • podtytuł
    • email
    • informacje o githubie
    • oraz inne takie jak: Twitter, Google Plus, Delicious i Facebook Like

W następnym poście dowiemy się jak swojego octopressa wrzucić na githuba.
