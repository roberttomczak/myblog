Wrzucanie Octopressa Na Githuba

Jan 17th, 2012

Aby blog nie siedział tylko na komputerze w kilku krokach możemy wrzycic go na Github Pages.

Aby to zrobić należy posępować według kroków:

  • zakładamy nowe repozytorium na githubie
  • następnie w terminalu przechodzimy do głównego katalogu octopressa
  • tworzymy nową gałąź np. git checkout -b source
  • następnie musisz dodac remote’a na te repo, a dokładnie użyć komendy git remote add origin git@github.com:nazwa_uzytkownika/nazwa_repo
  • a następnie pchnąc na source’a git push origin source
  • po zakończeniu wpisujemy rvm rvmrc trust
  • później przeładowywujemy rvm komendą rvm reload
  • następnie wpisujemy rake setup_github_pages i wklejamy nasz adres repozytorium git@github.com:nazwa_użytkownika/nazwa_repo
  • rake install
  • rake generate
  • rake deploy

Po wykonaniu wszystkich kroków i zmianie wartości root na /nazwa_repo w config.yml octopress powinien pojawić się na stronie nazwa_uzytkownika.github.com/nazwa_repo. Trzeba pamiętać że po zmianie pliku config.yml trzeba znowu użyć komend rake generate i rake deploy.
